Google Sheets Uploader

A web-based application that allows users to upload an Excel file and automatically update the contents of a Google Sheet in one step.

Features

  • File Upload: Users can upload an Excel file (`.xlsx` or `.xls`) via a simple web interface.
  • Google Sheets Integration: The uploaded file replaces the content of a specified Google Sheet.
  • Automated Workflow: Streamlines the process of updating Google Sheets with new data.
  • Serverless Backend: Uses Netlify Functions to handle file processing and API interactions.

Installation

  1. Clone the repository: 
    git clone https://github.com/your-username/google-sheets-uploader.git
  2. Install dependencies: 
    npm install
  3. Set up environment variables by creating a .env file: 
    
GOOGLE_PRIVATE_KEY="your-private-key"
GOOGLE_CLIENT_EMAIL="your-client-email"
SPREADSHEET_ID="your-google-sheet-id"
  4. Run locally: 
    npm start

Usage

  1. Open the web page in your browser.
  2. Click the "Choose File" button and select an Excel file (`.xlsx` or `.xls`).
  3. Click "Upload".
  4. The app will process the file and replace the contents of the specified Google Sheet.

Deployment

This application is designed to be deployed on Netlify using serverless functions. Follow these steps:

  1. Push your code to a GitHub repository.
  2. Connect your repository to Netlify.
  3. Configure the build settings:
    • Build Command: npm install
    • Publish Directory: public
  4. Set environment variables in Netlify for GOOGLE_PRIVATE_KEY, GOOGLE_CLIENT_EMAIL, and SPREADSHEET_ID.
  5. Deploy and test the live site.

Built With ❤️ Using

  • Express.js: Backend framework.
  • Multer: Middleware for handling file uploads.
  • XLSX: Library for parsing Excel files.
  • Google APIs Node.js Client: For interacting with the Google Sheets API.

Limitations

  • File Size: The maximum file size is approximately 4 MB due to Netlify's request size limit.
  • Execution Time: Netlify Functions have a maximum execution time of 10 seconds (free plan) or 15 seconds (paid plan).
  • Google Sheets API Limits: Ensure the total number of cells in your Google Sheet does not exceed 5 million.
